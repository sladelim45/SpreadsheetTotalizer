Spreadsheet Totalizer

This program allows users to obtain a total breakdown for one or multiple spreadsheets. It can be used for calculating integer or decimal totals by detecting a unique key (ex. name) and number values (ex. hours worked, customers served, etc.). Real life use cases are any type of scheduling or work-hour reporting spreadsheet where payroll specialists need to process payroll data or calculate earnings of employees.
